# Tetris MVP

브라우저에서 바로 실행할 수 있는 간단한 프론트엔드 테트리스 애플리케이션입니다.  
별도 설치나 빌드 없이 `index.html`을 열면 실행됩니다.

## 배포 링크

- 플레이하기: https://ryuboutintin.github.io/tetrisRepository/

## 실행 방법

1. 프로젝트 폴더로 이동합니다.
2. `index.html` 파일을 브라우저에서 엽니다.

## 포함 기능

- 10x20 테트리스 보드
- 블록 자동 낙하
- 좌우 이동
- 아래로 빠르게 내리기
- 블록 회전
- 줄 삭제
- 점수 표시
- 게임 오버 처리
- 재시작 버튼 및 `Enter` 키 재시작

## 조작법

- `←` / `→` : 좌우 이동
- `↓` : 빠르게 내리기
- `↑` / `Space` : 회전
- `Enter` : 재시작

## 디자인 방향

- 브라우저에서 바로 열 수 있는 정적 단일 페이지 구성
- 점수판과 조작 안내를 분리한 직관적인 레이아웃
- 캔버스 기반 게임 보드
- 모바일에서도 깨지지 않도록 반응형 레이아웃 적용

## 구현 계획

1. 정적 웹 구조를 먼저 만든다.  
   `index.html`, `style.css`, `script.js`만으로 실행되게 구성한다.

2. MVP 범위를 최소 기능으로 고정한다.  
   게임 보드, 블록 이동, 회전, 점수, 줄 삭제, 게임 오버, 재시작까지만 포함한다.

3. UI를 단순하지만 완성도 있게 설계한다.  
   정보 패널과 게임 보드를 분리해 초보자도 바로 이해할 수 있게 만든다.

4. 게임 로직을 순수 JavaScript로 구현한다.  
   보드 배열, 블록 생성, 충돌 판정, 회전, 줄 삭제, 점수 반영을 직접 처리한다.

5. 입력 처리와 게임 루프를 연결한다.  
   방향키 입력과 `requestAnimationFrame` 기반 낙하 업데이트를 연결한다.

6. 브라우저 직접 실행 환경에 맞게 마무리한다.  
   외부 의존성 없이 구성하고, 키 입력 시 페이지 스크롤도 방지한다.

## 파일 구성

- `index.html` : 화면 구조
- `style.css` : 레이아웃 및 스타일
- `script.js` : 테트리스 게임 로직

## 검증

- `node --check script.js`로 자바스크립트 문법 검증 완료
- 브라우저 렌더링은 사용자가 `index.html`을 직접 열어 확인 가능
