# Tetris MVP

브라우저에서 바로 실행할 수 있는 정적 단일 페이지 테트리스 애플리케이션입니다.  
별도 설치나 빌드 없이 `index.html`을 열면 랜딩 화면부터 바로 실행할 수 있습니다.

## 실행 방법

1. 프로젝트 폴더로 이동합니다.
2. `index.html` 파일을 브라우저에서 엽니다.

## 포함 기능

- 랜딩 화면에서 플레이 방법과 점수 규칙 안내
- `Play` 버튼 또는 `Enter` 키로 게임 시작
- 10x20 테트리스 보드
- 블록 자동 낙하
- 좌우 이동
- 아래로 빠르게 내리기
- 블록 회전
- 줄 삭제 및 점수 반영
- 게임 오버 오버레이 표시
- 재시작 버튼 및 `Enter` 키 재시작
- Web Audio API 기반 BGM 재생
- BGM 음소거 토글 버튼

## 화면 흐름

1. 첫 화면에서 게임 규칙과 조작법을 확인합니다.
2. `플레이 시작` 버튼 또는 `Enter` 키로 게임 화면으로 진입합니다.
3. 게임 시작과 함께 BGM이 재생됩니다.
4. 게임 오버 시 오버레이가 표시되며 버튼 또는 `Enter` 키로 즉시 재시작할 수 있습니다.

## 조작법

- 랜딩 화면: `Enter`로 게임 시작
- 게임 화면: `←` / `→` 로 좌우 이동
- 게임 화면: `↓` 로 빠르게 내리기
- 게임 화면: `↑` / `Space` 로 회전
- 게임 화면: `Enter` 로 재시작
- 게임 화면: `BGM 음소거` 버튼으로 사운드 토글

## 점수 규칙

- 1줄 제거: `100점`
- 2줄 제거: `300점`
- 3줄 제거: `500점`
- 4줄 제거: `800점`

## 디자인 방향

- 브라우저에서 바로 열 수 있는 정적 단일 페이지 구성
- 랜딩 화면과 게임 화면을 분리한 2단계 흐름
- 점수판과 조작 안내를 분리한 패널형 레이아웃
- 캔버스 기반 게임 보드
- 모바일에서도 깨지지 않도록 반응형 레이아웃 적용
- 따뜻한 아케이드 톤의 배경 그래디언트와 카드 UI

## 구현 요약

1. `index.html`, `style.css`, `script.js`만으로 동작하는 정적 구조로 구성했습니다.
2. 랜딩 화면과 게임 화면을 분리하고 클래스 토글로 전환합니다.
3. 게임 로직은 순수 JavaScript로 구현했습니다.
4. `requestAnimationFrame` 기반 루프와 키보드 입력을 연결했습니다.
5. Web Audio API로 간단한 테트리스풍 루프 BGM을 생성합니다.
6. 외부 의존성 없이 동작하며 키 입력 시 브라우저 스크롤을 방지합니다.

## 파일 구성

- `index.html` : 랜딩 화면과 게임 화면 구조
- `style.css` : 반응형 레이아웃과 패널 스타일
- `script.js` : 테트리스 게임 로직, 화면 전환, Web Audio BGM

## 검증

- `node --check script.js`로 자바스크립트 문법 검증 완료
- 브라우저 렌더링은 사용자가 `index.html`을 직접 열어 확인 가능
- 수동 확인 항목: 랜딩 화면 노출, 게임 시작 전환, 조작 입력, 게임 오버 오버레이, 음소거 버튼
