# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 프로젝트 개요

빌드 도구나 외부 라이브러리 없이 순수 HTML/CSS/JS로 구현된 자기소개용 Markdown 에디터입니다.
`index.html` 하나를 브라우저에서 바로 열어 사용합니다.

## 파일 구조

모든 코드(HTML, CSS, JS)가 `index.html` 단일 파일에 인라인으로 작성되어 있습니다.

## 주요 기능 및 구조

### 레이아웃
- **헤더**: 툴바 버튼 + 우측 액션 버튼(템플릿 불러오기, 복사, 저장)
- **사이드바**: 섹션 삽입 단축 버튼 8종 (기본 정보, 한 줄 소개, 기술 스택 등)
- **에디터 창** (`#editor`): `<textarea>`, 모노스페이스 폰트, Tab키 = 공백 2칸
- **드래그 구분선** (`#divider`): 에디터/미리보기 비율을 마우스로 조정 (20%~80%)
- **미리보기 창** (`#preview`): `parseMd()` 결과를 `innerHTML`로 렌더링
- **상태바**: 글자 수·줄 수·단어 수 실시간 표시, 자동 저장 상태 표시

### 커스텀 Markdown 파서 (`parseMd`)
외부 라이브러리 없이 정규식으로 구현. 처리 순서가 결과에 영향을 주므로 순서 변경 시 주의:
1. HTML 이스케이프 (`&`, `<`, `>`)
2. 펜스드 코드블록 (` ``` `)
3. 인라인 코드 (`` ` ``)
4. 헤딩 (H1~H6)
5. 인용문 (`>`)
6. 수평선 (`---`)
7. 굵게/기울임/취소선 (`**`, `*`, `~~`)
8. 이미지 → 링크 순 (이미지를 먼저 처리해야 링크 패턴과 충돌 방지)
9. 표 (GFM 스타일)
10. 순서 없는/있는 목록
11. 단락 처리 및 `<br>` 변환

### 자동 저장
`localStorage` 키 `md_intro`에 600ms 디바운스로 저장.
페이지 로드 시 저장된 내용이 있으면 복원하고, 없으면 기본 템플릿을 불러옴.

### 섹션 삽입 (`SECTIONS` 객체)
`SECTIONS` 객체에 섹션 키와 Markdown 템플릿 문자열이 매핑되어 있음.
새 섹션 추가 시 `SECTIONS`에 항목 추가 후 사이드바에 `<button>` 하나 추가하면 됨.

## 디자인 시스템

CSS 커스텀 속성(`:root`)으로 전체 색상 관리. 테마 변경은 `:root` 값만 수정:

| 변수 | 용도 |
|------|------|
| `--bg` | 페이지 배경 |
| `--surface` | 헤더·사이드바·미리보기 배경 |
| `--surface2` | 중첩 배경 |
| `--border` | 구분선 색상 |
| `--accent` | 주 강조색 (보라) |
| `--accent2` | 보조 강조색 |
| `--text` | 본문 텍스트 |
| `--muted` | 흐린 텍스트 |
