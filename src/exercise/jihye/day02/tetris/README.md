# Mini Tetris

브라우저에서 바로 실행되는 간단한 Tetris MVP입니다.

## 실행 방법

1. 이 폴더에서 `index.html`을 브라우저로 엽니다.
2. 별도 설치나 빌드가 필요 없습니다.

## 조작

- `←` `→`: 좌우 이동
- `↓`: 빠르게 내리기
- `↑`: 회전
- `Space`: 하드 드롭
- `R`: 재시작

## 구현 범위

- 점수 표시
- 라인 클리어
- 레벨 증가
- 게임 오버 처리
- 재시작

## 작업 방식

앞으로 이 프로젝트를 수정할 때는 다음 순서를 기본으로 합니다.

1. 구현 전에 먼저 방법을 짧게 설명합니다.
2. 필요한 변경 사항을 파일로 남깁니다.
3. `README.md` 같은 문서를 함께 갱신해서, 나중에 봐도 흐름이 남도록 합니다.

## 메모

고급 기능보다 MVP 동작을 우선합니다. 필요하면 다음 단계에서 모바일 조작, 다음 블록 미리보기, 사운드 등을 추가할 수 있습니다.
