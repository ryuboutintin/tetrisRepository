# Plan: Tetris MVP Web App

## Context
`src/exercise/bardroh/day02/tetris/` 디렉토리가 비어 있다. 브라우저에서 파일을 직접 열어 실행할 수 있는 단일 HTML 파일 형태의 테트리스 MVP를 만든다.

## 목표
- 외부 의존성 없이 브라우저에서 바로 실행 가능
- 점수 표시, 블록 이동/회전, 라인 클리어의 핵심 기능만 포함
- 깔끔하고 직관적인 UI

---

## 구현 계획

### 파일 구조
```
tetris/
├── index.html   — 마크업 구조만
├── style.css    — 스타일
└── game.js      — 게임 로직 전체
```
`index.html`에서 `<link rel="stylesheet" href="style.css">` 와 `<script src="game.js">` 로 연결.

### 레이아웃 (CSS)
- 배경: 어두운 계열 (#1a1a2e 등)
- 좌측: 게임 캔버스 (Canvas, 10열 × 20행, 셀 크기 30px → 300×600px)
- 우측 패널: 점수, 레벨, 라인 수, 다음 블록 미리보기, 조작법 안내

### 게임 로직 (JavaScript)

#### 테트리미노 (7종)
```
I, O, T, S, Z, J, L — 각각 색상 지정
```

#### 핵심 기능
1. **이동**: ← → ↓ (소프트 드롭)
2. **회전**: ↑ (시계방향 90°)
3. **하드 드롭**: Space
4. **자동 낙하**: setInterval, 레벨에 따라 속도 증가
5. **라인 클리어**: 꽉 찬 행 제거 후 위 블록 내려오기
6. **점수**: 클리어 라인 수 × 100 (2줄=300, 3줄=500, 4줄=800)
7. **레벨업**: 10라인마다 레벨 +1, 낙하 속도 증가
8. **게임 오버**: 블록이 최상단에 쌓이면 감지
9. **시작/재시작 버튼**

#### 렌더링
- `requestAnimationFrame` + Canvas 2D API
- 보드 배경 격자선 표시
- 현재 블록 + 고스트(낙하 위치 미리보기) 표시

### 검증 방법
> 외부 CSS/JS 파일을 참조하므로 `python3 -m http.server 8765` 로 서버를 띄운 뒤
> `http://localhost:8765/index.html` 에서 확인.

1. 서버 실행 후 브라우저에서 접속
2. 방향키로 블록 이동·회전 확인
3. Space로 하드 드롭 확인
4. 라인 클리어 시 점수 증가 확인
5. 게임 오버 후 재시작 버튼 동작 확인
