# 테트리스 (Tetris MVP)

브라우저에서 바로 실행 가능한 클래식 테트리스 게임입니다.

## 실행 방법

```bash
cd src/exercise/heesung33/day02/tetris
python3 -m http.server 8765
```

브라우저에서 `http://localhost:8765/index.html` 접속

## 파일 구조

| 파일 | 설명 |
|------|------|
| `index.html` | 게임 UI 레이아웃 (캔버스, 사이드 패널, 게임오버 오버레이) |
| `style.css` | 다크 테마 스타일링 |
| `game.js` | 게임 로직 (블록 생성, 이동, 회전, 충돌 판정, 점수 계산) |

## 기능 목록

### 핵심 기능

| 기능 | 설명 |
|------|------|
| 블록 생성 | 7종 표준 테트로미노(I, J, L, O, S, T, Z) 랜덤 생성 |
| 블록 이동 | 좌/우 이동 및 아래로 낙하 |
| 블록 회전 | 시계 방향 90도 회전 (충돌 시 회전 불가) |
| 즉시 낙하 | 스페이스바로 블록을 바닥까지 즉시 드롭 |
| 줄 제거 | 가로 한 줄이 꽉 차면 해당 줄 제거 |
| 충돌 판정 | 벽, 바닥, 기존 블록과의 충돌 감지 |
| 게임 오버 | 블록이 보드 상단을 넘으면 게임 종료 |

### 점수 시스템

| 동작 | 점수 |
|------|------|
| 소프트 드롭 (↓) | 1줄당 1점 |
| 하드 드롭 (Space) | 1줄당 2점 |
| 1줄 제거 | 100 x 레벨 |
| 2줄 동시 제거 | 300 x 레벨 |
| 3줄 동시 제거 | 500 x 레벨 |
| 4줄 동시 제거 (테트리스) | 800 x 레벨 |

### 레벨 시스템

- 10줄 제거마다 레벨 1 증가
- 레벨이 오를수록 블록 낙하 속도 증가
- 최소 낙하 간격: 100ms

### UI 기능

| 기능 | 설명 |
|------|------|
| 점수 표시 | 실시간 점수 업데이트 |
| 레벨 표시 | 현재 레벨 표시 |
| 다음 블록 미리보기 | 사이드 패널에 다음 블록 표시 |
| 고스트 블록 | 현재 블록의 착지 예상 위치를 반투명으로 표시 |
| 게임 오버 화면 | 최종 점수 표시 및 재시작 버튼 |
| 일시정지 | P키로 게임 일시정지/재개 |

## 조작법

| 키 | 동작 |
|----|------|
| `←` / `→` | 블록 좌우 이동 |
| `↑` | 블록 회전 |
| `↓` | 소프트 드롭 (1칸씩 빠르게 낙하) |
| `Space` | 하드 드롭 (즉시 착지) |
| `P` | 일시정지 / 재개 |

## 기술 스택

- HTML5 Canvas
- Vanilla CSS (다크 테마)
- Vanilla JavaScript (외부 라이브러리 없음)
