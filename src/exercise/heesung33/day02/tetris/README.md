# 테트리스 (Tetris)

브라우저에서 바로 실행 가능한 클래식 테트리스 게임입니다. 순수 HTML/CSS/JavaScript로 구현하였으며 외부 라이브러리를 사용하지 않습니다.

## 배포 URL

https://heesung33.github.io/vibecoding_test/index.html

## 실행 방법 (로컬)

```bash
cd src/exercise/heesung33/day02/tetris
python3 -m http.server 8765
```

브라우저에서 `http://localhost:8765` 접속 → 랜딩 페이지 → "게임 시작" 클릭 → 게임 플레이

## 파일 구조

| 파일 | 설명 |
|------|------|
| `index.html` | 랜딩 페이지 (조작법, 점수 계산, 블록 종류 안내 + 게임 시작 버튼) |
| `game.html` | 게임 플레이 페이지 (캔버스 기반 테트리스 보드 + 사이드 패널) |
| `game.js` | 게임 로직 (보드 관리, 블록 생성/회전/낙하, 줄 클리어, 점수/레벨 시스템) |
| `music.js` | Web Audio API 기반 BGM (Korobeiniki 테마, square wave 멜로디 + triangle wave 베이스) |
| `style.css` | 전체 스타일 (애니메이션 배경, 랜딩/게임 페이지 레이아웃, 반응형) |

## 기능 목록

### 핵심 기능

| 기능 | 설명 |
|------|------|
| 블록 생성 | 7종 표준 테트로미노(I, J, L, O, S, T, Z) 랜덤 생성 |
| 블록 이동 | 좌/우 이동 및 아래로 낙하 |
| 블록 회전 | 시계 방향 90도 회전 (충돌 시 회전 불가) |
| 소프트 드롭 | ↓키로 1칸씩 빠르게 낙하 (+1점/칸) |
| 하드 드롭 | 스페이스바로 블록을 바닥까지 즉시 드롭 (+2점/칸) |
| 줄 제거 | 가로 한 줄이 꽉 차면 해당 줄 제거 |
| 충돌 판정 | 벽, 바닥, 기존 블록과의 충돌 감지 |
| 게임 오버 | 블록이 보드 상단을 넘으면 게임 종료 |

### 점수 시스템

| 동작 | 점수 |
|------|------|
| 소프트 드롭 (↓) | 1칸당 1점 |
| 하드 드롭 (Space) | 1칸당 2점 |
| 1줄 제거 | 100 x 레벨 |
| 2줄 동시 제거 | 300 x 레벨 |
| 3줄 동시 제거 | 500 x 레벨 |
| 4줄 동시 제거 (테트리스) | 800 x 레벨 |

### 레벨 시스템

- 10줄 제거마다 레벨 1 증가
- 레벨이 오를수록 블록 낙하 속도 증가 (`max(100ms, 800 - (level-1)*80)`)
- 최소 낙하 간격: 100ms

### UI 기능

| 기능 | 설명 |
|------|------|
| 랜딩 페이지 | 조작법, 점수 계산, 블록 종류 안내 화면 |
| 점수 표시 | 실시간 점수 업데이트 |
| 레벨 표시 | 현재 레벨 표시 |
| 다음 블록 미리보기 | 사이드 패널 캔버스에 다음 블록 렌더링 |
| 고스트 피스 | 현재 블록의 낙하 예상 위치를 반투명으로 표시 |
| 게임 오버 화면 | 최종 점수 표시 및 재시작 버튼 |
| 일시정지 | P키로 게임 일시정지/재개 |
| BGM | Web Audio API 프로시저럴 BGM (BPM 140), 토글 버튼으로 on/off |
| 배경 애니메이션 | CSS 그라디언트 애니메이션 + 별 반짝임 + 그리드 패턴 |

## 조작법

| 키 | 동작 |
|----|------|
| `←` / `→` | 블록 좌우 이동 |
| `↑` | 블록 회전 |
| `↓` | 소프트 드롭 (1칸씩 빠르게 낙하) |
| `Space` | 하드 드롭 (즉시 착지) |
| `P` | 일시정지 / 재개 |

## 게임 설정

- 보드: 10x20 (COLS x ROWS)
- 블록 크기: 30px
- BGM: BPM 140, 게임 시작 시 자동 재생

## 기술 스택

- HTML5 Canvas
- CSS3 (그라디언트 애니메이션, 반응형 레이아웃)
- Vanilla JavaScript (외부 라이브러리 없음)
- Web Audio API (OscillatorNode 기반 프로시저럴 BGM)
