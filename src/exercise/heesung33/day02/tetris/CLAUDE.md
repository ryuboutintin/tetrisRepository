# CLAUDE.md — Tetris

## 프로젝트 개요

순수 HTML/CSS/JavaScript로 구현한 테트리스 게임. 외부 라이브러리 없음.

## 파일 구조

- `index.html` — 랜딩 페이지 (조작법, 점수 계산, 블록 종류 안내 + 게임 시작 버튼)
- `game.html` — 게임 플레이 페이지 (캔버스 기반 테트리스 보드 + 사이드 패널)
- `game.js` — 게임 로직 (보드 관리, 블록 생성/회전/낙하, 줄 클리어, 점수/레벨 시스템)
- `music.js` — Web Audio API 기반 BGM (Korobeiniki 테마, square wave 멜로디 + triangle wave 베이스)
- `style.css` — 전체 스타일 (애니메이션 배경, 랜딩/게임 페이지 레이아웃, 반응형)

## 실행 방법

```bash
cd src/exercise/heesung33/day02/tetris && python3 -m http.server 8765
```

브라우저에서 `http://localhost:8765` 접속. 랜딩 페이지 → "게임 시작" 클릭 → 게임 플레이.

## 게임 설정

- 보드: 10×20 (COLS × ROWS), 블록 크기 30px
- 속도: `max(100ms, 800 - (level-1)*80)` 간격으로 틱
- 레벨업: 10줄 클리어마다
- 점수: 1줄=100, 2줄=300, 3줄=500, 4줄=800 (× 레벨 배수)
- BGM: BPM 140, 게임 시작 시 자동 재생, 토글 버튼으로 on/off

## 조작

| 키 | 동작 |
|---|---|
| ← → | 좌우 이동 |
| ↑ | 회전 |
| ↓ | 소프트 드롭 (+1점/칸) |
| Space | 하드 드롭 (+2점/칸) |
| P | 일시정지 |

## 주요 구현 사항

- 고스트 피스: 현재 블록의 낙하 예상 위치를 반투명으로 표시
- 다음 블록 미리보기: 사이드 패널 캔버스에 렌더링
- 음악: Web Audio API의 OscillatorNode를 사용한 프로시저럴 BGM 생성 (외부 파일 불필요)
- 배경: CSS 그라디언트 애니메이션 + 별 반짝임 + 그리드 패턴
