## 계획 1. 테트리스 기능 개선

| 우선순위 | 기능 | 설명 |
|----------|------|------|
| 높음 | 최고점수 저장 | `localStorage`에 베스트 스코어 유지 |
| 높음 | 홀드 기능 | `C` 키로 현재 블록 보관/교환 |
| 중간 | 효과음 | Web Audio API로 줄 삭제·낙하 효과음 |
| 낮음 | 배경 음악 | 루프 BGM 추가 |
| 낮음 | 모바일 터치 | 스와이프/탭으로 조작 |

## 계획 2. 새 실습 추가

day02에 테트리스 외에 추가할 수 있는 미니 프로젝트 아이디어:

- **계산기** — 사칙연산 + 키보드 입력 지원
- **타이머 / 스톱워치** — 랩 기록 포함
- **숫자 맞추기 게임** — 1~100 랜덤 숫자 힌트 제공
- **색상 팔레트 생성기** — 랜덤 배색 + 클립보드 복사
