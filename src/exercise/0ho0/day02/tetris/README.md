# Tetris MVP

브라우저에서 바로 실행할 수 있는 순수 HTML/CSS/JavaScript 테트리스 웹앱입니다. 별도 서버, 빌드 도구, 외부 라이브러리 없이 `index.html` 파일만 열면 동작합니다.

## 실행 방법

1. 현재 폴더의 `index.html`을 브라우저로 엽니다.
2. 랜딩페이지에서 플레이어 이름을 입력합니다.
3. `Play` 버튼을 눌러 게임을 시작합니다.

## 주요 기능

- 랜딩페이지
  - 플레이 방법 안내
  - 점수 계산 안내
  - 플레이어 이름 입력
  - 랭킹 확인 버튼
- 테트리스 게임
  - 10x20 게임 보드
  - 7종 테트리스 블록
  - 자동 낙하
  - 블록 좌우 이동
  - 블록 회전
  - 빠른 하강
  - 다음 블록 미리보기
  - 줄 제거 및 점수 증가
  - 게임 오버 처리
- 랭킹
  - 게임 오버 시 플레이어 이름과 점수 저장
  - 브라우저 `localStorage`에 랭킹 저장
  - 점수 높은 순으로 최대 20개 표시

## 조작법

| 키 | 동작 |
| --- | --- |
| `ArrowLeft` | 왼쪽 이동 |
| `ArrowRight` | 오른쪽 이동 |
| `ArrowDown` | 빠른 하강 |
| `ArrowUp` | 회전 |
| `Space` | 시작 또는 재시작 |
| `Escape` | 랭킹 모달 닫기 |

## 점수 계산

| 제거한 줄 수 | 점수 |
| --- | ---: |
| 1줄 | 100점 |
| 2줄 | 300점 |
| 3줄 | 500점 |
| 4줄 | 800점 |

## 구현 메모

- 모든 UI, 스타일, 게임 로직은 `index.html` 안에 포함되어 있습니다.
- 랭킹 데이터는 서버가 아닌 현재 브라우저의 `localStorage`에 저장됩니다.
- 다른 브라우저나 시크릿 모드에서는 랭킹 데이터가 공유되지 않습니다.
