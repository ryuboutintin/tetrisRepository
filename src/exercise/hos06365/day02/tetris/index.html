Classic block puzzle

Mini Tetris

브라우저에서 바로 즐기는 간단한 테트리스 MVP입니다. 줄을 완성해 점수를 얻고, 블록이 천장까지 쌓이기 전에 최대한 오래 버티세요.

플레이 방법

  • ← / →: 블록 좌우 이동
  • ↓: 블록 한 칸 빠르게 내리기
  • ↑: 블록 회전
  • Space: 블록 즉시 드롭
  • P: 일시정지 또는 재개

점수 계산

  • 1줄 제거: 100점
  • 2줄 제거: 300점
  • 3줄 제거: 500점
  • 4줄 제거: 800점
  • 즉시 드롭: 내려간 칸마다 2점
